1. Overview

Symbol Handler is a palette tool that helps you insert, replace and manage Sketched Symbols in Inventor drawings while preserving position, scale, layer, leader and attribute values.

Open the palette from the ribbon tab MCG TOOLS → Symbol Handler (Drawing environment only).

2. Source (File / Local)

Switch between two symbol sources using the radio buttons at the top of the palette.

SourceBehavior
File Load symbols from a library .idw file (or a folder of .idw files). Type or paste a path into the box and press Enter, or click ⚙ to browse.
Local Show symbols already defined in the active drawing document.

3. Search & View Modes

  • Search: type to filter symbols by name (case-insensitive, matches substring).
  • View toggle (⊞ Grid / ☰ List): switch between thumbnail grid and compact list view.
  • Arrow keys (↑ / ↓): navigate the list when focused.
  • Double-click a symbol: start insert mode immediately.
  • Right-click a symbol: context menu → Insert into Drawing.

4. Properties & Attributes

When a symbol is selected on the palette, the Properties panel shows:

  • Scale and Rotate (degrees) — used when inserting new symbols.
  • Static — hide grip points (scale + rotate handles).
  • Symbol Clipping — trim annotations behind the symbol.
  • Leader — attach a leader line to the picked geometry.
  • Leader Visible — show/hide the leader line (only meaningful when Leader is on).
  • Attributes DataGrid — lists all prompt fields of the symbol definition with editable values. Populated automatically from the symbol's defaults; fill in custom values before Insert or Replace.

5. Replace Symbols

Replace single

  1. Select the new symbol from the palette.
  2. (Optional) Edit attribute values in the Attributes DataGrid.
  3. Click 🔄 Replace → cursor enters pick mode.
  4. Click an old symbol on the drawing → replaced in place.

Replace All

  1. Select the new symbol from the palette.
  2. Click 🔄 Replace All ▾ → choose scope:
    • Current Sheet Only — replace all matching instances on the active sheet.
    • All Sheets — replace everywhere in the document (confirmation dialog shows breakdown per sheet).
  3. Click an old symbol on the drawing → all instances sharing its definition are replaced in one Undo step.

What is preserved

PropertyBehavior
PositionAlways preserved.
ScaleAlways preserved.
LayerPreserved.
Static / Symbol Clipping / Leader VisiblePreserved from the old instance.
Leader (attached geometry)Re-attached to the same drawing curve / sketch line.
RotationPreserved when old and new definitions are the same (update case). Reset to 0 when definitions differ (cross-definition replace, to avoid flipped orientation).
Attribute valuesCopied from old instance when fields match. Values you type in the Attributes DataGrid override the copied values.
Cross-definition flip: Different symbol definitions may have different "natural up" orientations. If the old symbol was rotated 180° to look right, the new symbol would appear upside-down. Symbol Handler resets rotation to 0 on cross-def replace so the new symbol displays in its correct orientation. If you need to preserve rotation across different definitions, adjust manually with Inventor's built-in rotate handle.

6. Insert Symbol

  1. Select a symbol from the palette.
  2. (Optional) Edit Scale, Rotate, checkboxes, and attribute values.
  3. Double-click the symbol, or right-click → Insert into Drawing.
  4. Click on the drawing:
    • on an edge or sketch line → inserted with a leader attached to that geometry;
    • on empty space → inserted as a floating symbol (no leader).
  5. Continuous mode: another insert is primed automatically. Press Esc to exit.

7. Scan & Select Similar

Select Similar

  1. Click any symbol on the drawing to pick it. The ⎚ Select Similar button becomes enabled.
  2. Click Select Similar → all instances sharing that definition are highlighted on the active sheet, and the status bar shows the match count.

Scan Sheet

Click 🔍 Scan Sheet to highlight every symbol on the active sheet whose definition is not present in the current palette source. The status bar lists each unknown definition with its instance count, e.g. 5/41 NOT in palette (3 type(s)): thickness-right (3), WELD2 (1), CL DOOR (1).

Clear Highlight

Click ✖ Clear Highlight to clear the current selection set after Scan or Select Similar.

8. Edit Attributes In-Place

Click any symbol already on the drawing. The Attributes DataGrid loads that instance's actual prompt values. Edit a cell → the symbol updates immediately (each edit is one Undo step).

9. Tips & Troubleshooting

  • Undo / Redo: every Replace (single or All) and every Insert is a single Undo step. Use Ctrl+Z normally.
  • Palette hidden on startup: open it again from the ribbon button. The palette doesn't auto-restore on session reload.
  • Attributes empty when replacing a non-attribute symbol with an attribute symbol: fill the Attributes DataGrid first, then Replace — the typed values are applied to the new symbol.
  • Library not loading: verify the path in the File box (type or paste, then press Enter). Library documents are opened invisibly and closed automatically.
  • Log / debug output: open DebugView (Sysinternals) to watch internal messages prefixed with [SymbolReplaceService], [PaletteController], etc.
