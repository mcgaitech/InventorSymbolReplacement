MCG Symbol Handler

Autodesk Inventor 2023 · Drawing Add-In · v1.0.0

Symbol Handler is a palette tool for Autodesk Inventor 2023 that lets you insert, replace and manage Sketched Symbols in drawing documents (.idw). Open it from the MCG TOOLS tab in the Drawing environment.

System Requirements

ComponentRequired
Autodesk Inventor2023 (version 27.x or later)
Operating SystemWindows 10 / 11 (64-bit)
.NET Framework4.8 (pre-installed on Windows 10 v1903+)
Administrator rightsNot required

Installation

Package contents

MCG_InventorSymbolHandler_v1.0.0/
  ├─ MCG_InventorSymbolHandler.dll ← add-in library
  ├─ MCG_InventorSymbolHandler.addin ← Inventor descriptor
  ├─ Install_AutoLoadInventorAddin.bat ← installer script
  └─ Instructions.html ← this file

Steps

  1. Close Autodesk Inventor if it is currently running.
  2. Copy all three files below into C:\CustomTools\Inventor\ (create the folder if it does not exist):
    • MCG_InventorSymbolHandler.dll
    • MCG_InventorSymbolHandler.addin
    • Install_AutoLoadInventorAddin.bat
  3. Run Install_AutoLoadInventorAddin.bat (double-click it inside C:\CustomTools\Inventor\).
    The script scans the folder for *.addin files and copies them into %APPDATA%\Autodesk\Inventor 2023\Addins\ so Inventor will discover the add-in on startup. Press any key to close the script window when it shows Success.
  4. Open Autodesk Inventor 2023 and open any Drawing (.idw) file.
  5. Verify the add-in loaded:
    • Look for the MCG TOOLS tab in the ribbon, or
    • Go to Tools → Add-In Manager and confirm MCG_InventorSymbolHandler is listed and ticked as Loaded.
    If the tab does not appear, see Troubleshooting.
Adding more add-ins later: the same Install_AutoLoadInventorAddin.bat script works for any number of add-ins — drop more *.addin files into C:\CustomTools\Inventor\ and run the script again.
Uninstall: close Inventor, then delete MCG_InventorSymbolHandler.dll from C:\CustomTools\Inventor\ and MCG_InventorSymbolHandler.addin from %APPDATA%\Autodesk\Inventor 2023\Addins\.

Opening the Panel

  1. Open a Drawing (.idw) file in Inventor 2023.
  2. Click the MCG TOOLS tab in the ribbon.
  3. In the Drawing panel, click Symbol Handler.
  4. The Symbol Handler palette opens and docks to the side of the screen.
The add-in is only active in the Drawing environment. The MCG TOOLS tab will not appear when an Assembly or Part document is open.

Workflow at a Glance

The typical usage follows this sequence. Click any step to jump to its detailed section.

1 📂 Set Source Load a library file or use symbols from the active drawing
›
2 🔎 Find Symbol Search or browse the symbol list
›
3 ⚙️ Set Properties Scale, rotate, leader options & attribute values
›
4a ➕ Insert Place a new symbol on the drawing
│
4b 🔄 Replace Swap existing symbol(s) with the selected definition

Steps 4a and 4b are independent — use Insert to add new symbols, Replace to update existing ones.

Feature Map

All features available in the palette — click any card to read the full details.

➕ Insert Symbol Place a symbol on the drawing. Attach it to an edge or sketch line with a leader, or drop it as a floating annotation. Runs in continuous mode — press Esc to exit. 🔄 Replace (Single) Pick an existing symbol on the drawing and swap its definition with the one selected in the palette. Position, scale, leader and attributes are preserved. ⏩ Replace All Replace every instance of a definition in one click — scoped to the current sheet or the entire document. Completes as a single Undo step. ✏️ Edit Attributes In-Place Click any symbol on the drawing to load its attribute values into the grid. Edit a cell and press Enter — the symbol updates immediately. ⎚ Select Similar Pick a symbol on the drawing, then highlight all instances that share the same definition on the active sheet. Shows instance count in the status bar. 🔍 Scan Sheet Highlight every symbol on the active sheet whose definition is not in the current palette source. Useful for auditing outdated or unknown symbols.

Source

The SOURCE row at the top of the palette controls where the symbol list is populated from. The ? button on the right opens this guide.

ModeHow to use
File Type or paste a full file path into the path box and press Enter to load. Click ⚙ to open a file browser. The source must be a valid Inventor Drawing (.idw) that contains Sketched Symbol definitions.
Local Shows symbols defined inside the currently active drawing document. Switch to this mode when your symbol library is embedded in the drawing itself.

Properties

Shown when a symbol is selected in the palette. These values are applied on the next Insert. They are also preserved or overridden during Replace (see What Is Preserved).

FieldDescription

Rotate°Rotation in degrees counter-clockwise. Default: 0.
StaticHides scale/rotate grip handles on the symbol. Default: on.
ClippingTrims annotations behind the symbol boundary.
LeaderAttaches a leader line when placing on a drawing edge or sketch line.
Leader VisibleShows or hides the leader line (only relevant when Leader is on).

Attributes

The ATTRIBUTES grid below Properties lists all prompt fields defined in the symbol. The Field column is read-only; the Value column is editable.

  • Values entered here are applied when you Insert or Replace.
  • When you click a symbol already on the drawing, the grid loads that instance's live values for in-place editing.
  • If the symbol definition has no prompt fields, the grid is empty — this is normal.

Insert Symbol

  1. Select a symbol in the palette list.
  2. Adjust Properties (Scale, Rotate°, checkboxes) and fill in any Attribute values.
  3. Start insert mode:
    • Double-click the symbol, or
    • Right-click → Insert into Drawing.
  4. Click on the drawing:
    • On an edge or sketch line → symbol is inserted with a leader attached.
    • On empty space → symbol is inserted as a floating annotation (no leader).
  5. The tool stays active (continuous mode). Place more symbols or press Esc to exit.
Each insert is a single Undo step — Ctrl+Z reverses it.

Replace Symbol (Single)

  1. Select the new symbol definition from the palette.
  2. (Optional) Fill in Attribute values to apply to the replacement.
  3. Click 🔄 Replace (Pick Mode). The cursor enters pick mode.
  4. Click the existing symbol on the drawing you want to replace. It is swapped in-place.
The Replace button is grey until a symbol is selected in the palette list.

For details on what position, scale and attribute data is carried over, see What Is Preserved.

Replace All

Replaces every instance of a given definition at once, in a single Undo step.

  1. Select the new symbol from the palette.
  2. (Optional) Fill in Attribute values.
  3. Click Replace All ▾ and choose a scope:
    • Current Sheet Only — replaces all matching instances on the active sheet.
    • All Sheets — replaces across the entire document (a dialog shows the sheet-by-sheet count before confirming).
  4. Click any instance of the old symbol on the drawing. All matching instances are replaced immediately.
Replace All is one Undo step — pressing Ctrl+Z once restores all replaced instances.

Edit Attributes In-Place

Update the text values of a symbol that is already on the drawing, without going through Replace.

  1. Click directly on a symbol on the drawing.
  2. The Attributes grid loads that instance's current prompt values.
  3. Click a cell in the Value column and type a new value.
  4. Press Enter or click elsewhere — the symbol updates on the drawing immediately.
Each cell edit is its own Undo step.

Scan Sheet

Highlights all symbols on the active sheet whose definition is not found in the current palette source — useful for auditing outdated or mismatched symbols.

  1. Ensure the correct Source is loaded.
  2. Click 🔍 Scan Sheet.
  3. Unrecognised symbols are highlighted on the drawing. The status bar shows a breakdown:
    5/41 NOT in palette (3 type(s)): thickness-right (3), WELD2 (1), CL DOOR (1)
  4. Click ✖ Clear Highlight when done.

Select Similar

Find and highlight all instances of a specific symbol definition on the active sheet.

  1. Click any symbol directly on the drawing. The ⎚ Select Similar button enables.
  2. Click ⎚ Select Similar.
  3. All symbols sharing the same definition are highlighted. The status bar shows the count, e.g. 3 instance(s) of "WELD2" selected.
  4. Click ✖ Clear Highlight to deselect.
Select Similar operates on the active sheet only. Switch sheets and run it again to check other sheets.

What Is Preserved During Replace

PropertyBehavior
Position (insert point)Always preserved exactly.
ScaleAlways preserved.
LayerPreserved.
Static / Clipping / Leader VisiblePreserved from the old instance.
Leader (attached geometry)Re-attached to the same drawing curve or sketch line.
Rotation Same definition (update): rotation is preserved.
Different definition (cross-replace): rotation is reset to 0° to avoid the new symbol appearing flipped due to a different native orientation.
Attribute values Copied from the old instance where field names match. Values entered in the Attributes grid override the copied values.
Why rotation resets on cross-definition replace: If symbol A was rotated 180° to look correct and you replace it with symbol B, keeping 180° would flip B upside-down because the two definitions have different natural orientations. Adjust rotation manually with Inventor's rotate handle if needed after a cross-definition replace.

Troubleshooting

MCG TOOLS tab is not visible in the ribbon

  • Open Tools → Add-In Manager and check that MCG_InventorSymbolHandler is listed and ticked as Loaded.
  • If the entry is missing, verify that all three files exist in C:\CustomTools\Inventor\ and re-run Install_AutoLoadInventorAddin.bat, then restart Inventor.
  • The tab only appears in the Drawing environment — open a .idw file first.

Symbol list is empty after setting a File source

  • Confirm the path in the Source box is correct (type or paste, then press Enter).
  • The file must be a valid .idw drawing that contains Sketched Symbol definitions.

Attributes are empty after replacing a non-attribute symbol with an attribute symbol

Fill the Attributes grid before clicking Replace. Values you type are applied to the new symbol even when the old one had no attributes.

Symbol appears flipped or rotated incorrectly after Replace

This is expected on cross-definition replaces — rotation resets to 0°. See What Is Preserved for explanation. Adjust manually with Inventor's rotate handle.

Replace / Replace All button stays grey

Select a symbol in the palette list first. The button enables only when a definition is selected.

Panel does not reopen after closing

Click MCG TOOLS → Symbol Handler in the ribbon again. The palette does not auto-restore when Inventor is restarted.
